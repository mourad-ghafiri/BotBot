# Search

Search the web, news, and images; scrape pages; extract structured data.

## Tools

- **search_web** — Search the web using DuckDuckGo
- **search_news** — Search news using DuckDuckGo
- **search_images** — Search images using DuckDuckGo
- **search_scrape** — Scrape a webpage and extract text content
- **search_extract** — Extract elements from a page using a CSS selector
- **search_links** — Extract all links from a webpage

## Usage

- Start with `search_web` for most lookups. Use `search_news` for current events.
- Use `search_scrape` to read article/page content. Use `search_extract` with a CSS selector for structured data (prices, rows, headings).
- Use `search_links` to discover download URLs or crawl a site.
- Search tools fetch static HTML only. If you need JS rendering, login, or interaction — use the browser skill instead.
